QuickHeaders (Add in for MS Outlook 2010)

by Jimbo Esposito 04.01.12

Allows quick access to e-mail message headers from within the MS Outlook inspector. Simply right-click and choose "Show Headers" on any message, without having to select it or read it first.

Coded in C# using Microsoft Visual Studio 2010 Project Type: Outlook 2010 Add-In
